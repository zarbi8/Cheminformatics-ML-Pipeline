Automated Cheminformatics & Machine Learning Research Pipeline 🧪💻An end-to-end, multi-phase automated workflow designed for molecular property prediction and Organic Photovoltaic (OPV) material discovery.🚀 Repository Structure & PhasesPhase 1: Core Cheminformatics & Machine Learningcheminformatics_ml_pipeline.py: The core engine. Extracts 1D/2D descriptors and Morgan FPs, trains Random Forest & XGBoost models, and performs virtual screening.generate_structures.py: A modular script to automatically convert SMILES strings into 2D structural .png images.Phase 2: Explainable AI (XAI)phase2_shap_explainability.py: Implements SHAP to interpret the XGBoost models. Breaks down the "black box" to explain which molecular features drive the property predictions.Phase 3: Advanced AI Architecturesphase3_streamlit_webapp.py: A functional Web App. Users input SMILES to visualize structures and get real-time property predictions.phase3_gnn_pytorch.py: A Deep Learning template using PyTorch Geometric for Graph Convolutional Networks (GCN).🛠️ Installationgit clone [https://github.com/yourusername/cheminformatics-ml-pipeline.git](https://github.com/yourusername/cheminformatics-ml-pipeline.git)
